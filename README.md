# portfolio-rebalancing

## 배경, 필요성 정리
-> 포트폴리오 투자 필요(장점), 개인별로 선호도 투자철학이 있어서 나만의 포트폴리오 구성 원함 -> 포트폴리오 투자가 의미있으려면 비중 조절이 필요하나 현실적으로 직장인들은 어려움 -> AI기반으로 자동 리밸런싱해주는 서비스 만들어줌

기존의 포트폴리오 서비스가 있는데 우리의 강점은?

- 매일 정기적으로 리밸런싱
- 사용자에게 리포트 해줄게
- 실시간으로 발생하는 뉴스 정보들 바로 적용해서 수익률

### 에이전트 종류
**- 뉴스 에이전트 (시그널 에이전트)** 
    
    정성 데이터 수집 후 정형 데이터로 변환하여 판단
    
    1안. 있는거 해보기
    
    2안. 새로운 아이디어로
    
- 수익 에이전트 (해보고 결정)
- 비용 에이전트 (펀드매니저) 
    
    세금 + 수수료
    
- 판단 에이전트(마스터) + 검증
